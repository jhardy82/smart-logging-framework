# PowerCompany-smart-logging-framework

Intelligent logging framework with context-aware filtering

## 📋 Overview

This tool was extracted from the PowerCompany automation framework for standalone use and distribution.

**Category**: Development Tools
**Main Script**: SmartLogger.ps1

## 🚀 Installation

```powershell
# Clone the repository
git clone https://github.com/jhardy82/PowerCompany-smart-logging-framework.git

# Navigate to the repository
cd PowerCompany-smart-logging-framework

# Import the main module
. .\src\SmartLogger.ps1
```
